# Assignment
### Career & Skills Development Assistant Apps
#### made in Python

1. Goal Tracker（职业目标追踪）
	made by Nicholas
	要做的功能：
		- 输入目标（短期、长期）
		- 选择目标类型（例如：技能、证书、职位提升）
		- 设置 deadline
		- 设置进度（0–100%）
		- 显示目标列表（例如：用 listbox / table 展示）
		- 可以更新、删除目标
	额外可选：
		- 进度条
		- 快要到期的目标提醒

2. Skills Log（技能记录）
	made by Kar Wai
	要做的功能：
		- 添加已完成课程
		- 添加证书（名称、机构、日期）
		- 添加技能（如：Python、SQL、Teamwork）
		- 以列表/table 显示
		- 删除或编辑记录

3. Daily Task Planner（每日任务 / 学习安排）
	made by See Ze
	要做的功能：
		- 添加一天的任务
		- 可设置日期
		- 记已完成/未完成
		- 可过滤 “今天”、“明天”、“本周”
		- 显示任务列表（checkbox）
	加分功能（可选）：
		- 简单日历（Python 自带 calendar 模块）
		- 今日学习提醒弹窗

4. Achievement Summary & Interview Prep（成就摘要 + 面试准备）
	made by junyu
	要做的功能：
		- Achievement Summary
		- 自动读取 Goal Tracker 进度
		- 自动统计 Skills Log 完成数量
		- 自动统计 Task Planner 完成度
		- 生成 Summary（文本或简单 PDF）
	Interview Tips（问答库）:
		- 用户可以储存 “面试问题 & 自己的答案”
		- 可以新增、编辑、删除
		- 随机抽一个 Q&A 作为练习
